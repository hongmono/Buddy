# Buddy 👻

macOS 화면 위에 사는 귀여운 컴패니언. 여러 마리의 캐릭터가 화면을 자유롭게 떠다니며, 상황에 맞는 말을 걸어주고, 다양한 방식으로 교감할 수 있어요.

## 기능

### 캐릭터
4종의 빌트인 벡터 캐릭터와 커스텀 PNG 이미지를 지원합니다.

| 캐릭터 | 설명 |
|--------|------|
| 👻 유령 | 기본 캐릭터. 둥실둥실 떠다니는 물결 하단 |
| 🐱 고양이 | 삼각 귀, 세로 동공, 수염, 흔들리는 꼬리 |
| 🟢 슬라임 | 물방울 형태, 찌그러지는 탄성 애니메이션 |
| ☁️ 구름 | 뭉게구름 형태, 볼터치, 부풀어오르는 애니메이션 |
| 🖼️ PNG | 사용자가 직접 PNG 이미지를 불러와 사용 (정적) |

- 최대 5마리까지 동시에 화면에 띄울 수 있음
- 각 캐릭터별 이름, 성격, 크기(0.5x~3.0x) 커스터마이즈 가능
- 빌트인 캐릭터는 감정 표현, 눈 추적, 깜빡임 애니메이션 지원

### 이동
- 화면을 자유롭게 부유하며 떠다님
- 위아래로 살랑살랑 흔들리는 bobbing 애니메이션
- 속도가 자연스럽게 변하고, 가끔 멈춰서 생각하는 듯한 모습
- 화면 가장자리에 닿으면 튕겨서 방향 전환

### 눈 & 감정
- 마우스 커서를 따라 눈동자가 움직임
- 4가지 감정 상태: idle(나른), happy(기쁨), surprised(놀람), sleepy(졸림)
- 랜덤 눈 깜빡임
- 오래 가만히 두면 졸린 표정으로 변하며 "zzZ..." 말풍선

### 인터랙션
| 동작 | 반응 |
|------|------|
| 클릭 | 놀란 반응 |
| 더블클릭 | 고정 해제, 다시 돌아다님 |
| 트리플클릭 | "그만 찔러!!" 화남 반응 |
| 쓰다듬기 (좌우 드래그) | "좋아좋아~" 기쁨 반응 |
| 꾹 누르기 | "깜짝이야!" 놀람 반응 |
| 드래그 | 원하는 위치로 이동 |
| 마우스 올리기 | 기쁨 표정으로 변경 |

### AI 대화
- 로컬 [Claude Code](https://claude.ai/claude-code) CLI를 통해 AI 대화
- 캐릭터별 독립된 AI 성격 설정
- 시간대, 앱 전환, 유휴 상태에 따라 자동으로 말풍선 표시
- 클릭하면 채팅창을 열어 직접 대화 가능
- Claude CLI가 없어도 시간대별 인사, 랜덤 대사 등 fallback 제공

### 설정
- 캐릭터 추가/편집/삭제 (최대 5마리)
- 캐릭터별 이름, 성격, 외형, 크기 설정
- 말풍선 빈도 조절 (3~30분)
- 로그인 시 자동 시작
- Claude CLI 연결 상태 확인

## 요구사항
- macOS 14.0+
- (선택) [Claude Code](https://claude.ai/claude-code) CLI 설치 시 AI 대화 가능

## 빌드
Xcode에서 `Buddy.xcodeproj`를 열고 빌드/실행.
